Purpose

Electronic Return Processing eliminates the need to handle paper and, using an automated process, reduces the time to load report and return data. This allows the Department to reconcile or settle a return or report to a corresponding payment quickly and efficiently.

Activities

The electronic return process begins when enrolled, un-enrolled, obligated or voluntary taxpayers file returns or reports by electronic means (e.g., vendor supported web applications, DOR web applications, import functionality, EDI, XML, or MeF). Taxpayers who file using approved vendor web applications or DOR web applications provide information by data entry. Import functionality is available in several of the DOR vendor applications as an alternative method, which is a good choice for individuals who have numerous tax schedules or large amounts of data to transmit.
