Purpose

When a paper check is received from an unregistered entity, a taxpayer or employer account is created, which allows the payment and return to be processed in SUNTAX.

Activities

Check Processing begins with the receipt of an image of a paper check for an entity that is not currently registered with the Department. To process a payment in SUNTAX, an account must exist. Using the information on the check, an account is created by data capturing the entity name, address, and telephone number. When possible, the entity type (e.g., corporation, partnership, etc.) is assigned to the account, along with the determined tax obligations and liabilities. The process ends when all required data has been accurately entered, an account properly created and/or obligated for tax, and account information is viewable and electronically retrievable.
