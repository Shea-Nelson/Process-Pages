Purpose

Check Processing deposits paper payments into the State Treasury, as mandated by state law to fund governmental operations. Primary external customers are the Treasury, other state agencies and local governments.

Activities

Check processing begins with the receipt of checks (including money orders) and associated payment coupons or tax documents. Incoming checks and documents are sorted by tax and document type, then forwarded to the Screening Unit. Screening ensures the payment coupons have the appropriate information needed for posting to the correct account, tax type, and applied period. Screeners batch like-remittances, print web header sheets for each batch, and send all coupons and checks to the Validations Unit for processing. All back-up documentation for each like-remittance is sent to the imaging center for processing in Document Processing. The Validations Unit runs the remittance coupons and checks through high-speed equipment that images both sides of the documents. Using image recognition software and/or manual data entry, the Validation Unit perfects the information and generates a payment file, which is sent to the SUNTAX system for posting to taxpayer accounts. When the file is successful, SUNTAX sends a confirmation that automatically starts a process to prepare a file of deposit data and check images for electronic transmission to the bank for deposit. The checks are stored for a 30-day period after the electronic file has been transmitted to the bank for deposit, after which the checks are destroyed. Once the payments are completed by the Return Reconciliation Process, the coupons and documents are destroyed, and the images of them become the document of record. This ends the process.
